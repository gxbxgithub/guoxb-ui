# guoxb-ui

> Mobile UI elements for vue.js

## Build Setup

``` bash
# install
npm install

# dev
npm run dev

# guoxb-ui build
npm run dist
```

## guoxb-ui Setup

``` bash
# install
npm install guoxb-ui --save

# Vue.use()
import GuoxbUI from 'guoxb-ui'
Vue.use(GuoxbUI)

# use components
<g-button :round="true" type="normal">按钮</g-button>
```
